# Unstructured Data Analytics — NLP & LLM Pipeline Portfolio

Graduate coursework portfolio from MISY436: Unstructured Data Analytics 
at the University of Delaware (Spring 2026).

This repository contains Python notebooks covering the full NLP and 
LLM pipeline — from raw text extraction to classification, 
topic modeling, and multi-model API integration.

---

## Key Notebooks

### Exercise2.ipynb — Paragraph-Level Text Extraction Pipeline
Extracts and analyzes paragraph-level text from 5,352 SEC regulatory 
letters using Python and Polars. Processes structured metadata including 
document headings, file identifiers, and paragraph text for downstream 
NLP analysis. Directly replicable for policy document annotation pipelines.

### Exercise_4_Gemini_API_Calls.ipynb — LLM API Integration
Implements Gemini API calls for text analysis tasks, demonstrating 
prompt engineering, structured JSON output parsing, and result 
formatting — applicable to multi-model benchmarking pipelines.

### Assignment_3.ipynb — LDA & BERTopic Topic Modeling
Applies LDA and BERTopic to JPMorgan earnings call transcripts using 
spaCy, CountVectorizer, and sklearn. Evaluates optimal topic counts 
using coherence and perplexity metrics across Management Discussion, 
Q&A, and Answer sections.

### Assignment_2.ipynb — Text Classification & Sentiment Analysis
Builds text classification and sentiment analysis pipelines using 
Python NLP libraries — covering text representation, feature 
extraction, and classification evaluation.

### Assignment_4.ipynb — Recommender Systems & Social Network Analysis
Implements content-based and collaborative filtering recommender 
systems using cosine similarity, user-item matrix analysis, 
and social network graph methods.

### 11_2_Social_Networks.ipynb — Social Network Analysis
Graph-based network analysis using Python — node relationships, 
centrality measures, and community detection.

### 12_2_Computer_Vision.ipynb — CNN & YOLO Image Classification
Image classification using YOLO and CNNs — object detection 
and visual recognition pipelines.
